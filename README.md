# 01 Python Basics

This is my first Python project as part of my AI learning journey.

## Project: Simple Calculator

This project is a command-line calculator built with Python.

The calculator allows the user to:

- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers
- Handle division by zero
- Handle invalid number inputs
- Perform multiple calculations without restarting the program

- Git

- VS Code

## Python concepts practiced

In this project, I practiced:

- Variables
- User input
- Type conversion with `float()`
- Conditional statements with `if`, `elif`, and `else`
- Error handling with `try` and `except`
- Loops with `while True`
- Functions
- Returning values with `return`
- Git branches
- Pull requests
- Merging changes into `main`

## How to run the project

Open the terminal inside the project folder and run:

```bash
python3 calculator.py
```

## Example usage

```text
Simple Calculator
Enter the first number: 10
Enter operator (+, -, *, /): +
Enter the second number: 5
The result is: 15.0
Do you want to calculate again? (yes/no): yes
```

Example with division by zero:

```text
Enter the first number: 10
Enter operator (+, -, *, /): /
Enter the second number: 0
Error: You cannot divide by zero.
```

Example with invalid input:

```text
Enter the first number: abc
Error: Please enter valid numbers.
```
